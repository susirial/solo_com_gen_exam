# 边界与限制

## 本 Skill 不做什么

- 不直接依赖当前仓库中的 `paper/` 目录代码
- 不假设 `contracts.paper`、`core.config`、模板目录或 API 路由存在
- 不默认调用任何外部大模型接口
- 不内置数据库、Web 服务或 PDF 渲染链路

## 使用限制

- 若用户没有提供完整输入，优先追问，不要臆造关键约束
- 若用户只需要知识点讲解，不应触发本 Skill
- 若用户只要调试代码实现，应转到代码处理流程，而不是命题流程

## 风险提示

- 试卷内容质量仍取决于上游生成质量
- HTML 模板负责版式，不负责纠错
- 若输入结构不合法，应先修正结构，再渲染页面
