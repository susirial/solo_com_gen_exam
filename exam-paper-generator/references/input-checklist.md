# 输入确认清单

## 整卷级信息

- 学段：小学 / 初中 / 高中
- 年级
- 学科
- 试卷类型：练习卷 / 单元卷 / 月考试卷 / 期中 / 期末 / 模拟卷
- 总分
- 考试时长
- 使用场景：课堂练习 / 家庭作业 / 正式考试 / 讲评课

## 题目与分节信息

- 分几节
- 每节题型
- 每节题量
- 每题分值
- 每节总分
- 是否有共享材料

## 内容约束

- 知识点范围
- 难度要求
- 是否强调某些题型
- 是否避开某些知识点或出法
- 是否已有参考题或教材信息

## 输出要求

- 是否需要标准答案
- 是否需要解析或评分要点
- 是否需要教师版 HTML
- 是否需要学生版 HTML
- 是否需要结构化 JSON

## 单题重生 / 编辑时补充确认

- 目标题号
- 所属分节
- 动作类型：重生 / 改写 / 补解析 / 调难度 / 翻译
- 保留条件
- 目标难度
- 教师备注
