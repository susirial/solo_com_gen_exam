# HTML 渲染说明

本 Skill 使用 Jinja2 模板渲染教师版和学生版 HTML。

## 输入

渲染脚本接受一份符合 `schema.md` 的 JSON 文件。

## 模板

- `assets/templates/teacher.html.j2`
- `assets/templates/student.html.j2`

## 教师版

教师版页面应展示：

- 试卷标题和元信息
- 分节标题与说明
- 题目内容
- 标准答案
- 解析或评分要点
- 分值与知识点

## 学生版

学生版页面应展示：

- 试卷标题和元信息
- 分节标题与说明
- 题目内容
- 选项或作答区域

学生版不展示：

- 标准答案
- 解析
- 知识点提示

## 模板原则

- 模板只做展示层逻辑
- 所有结构修正和默认值补齐都应在 Python 脚本中完成
- 两个版本必须共用同一份试卷数据源
